## Event Planner App

An event planner application built with React and Firebase

## Demo

https://hannakozak.github.io/food-order-app/

## Build with

- React
- React Hooks
- React Router
- Styled Components

## Features

- Using the Firebase Realtime Database as database and storage.
- Ordering and basket functionality.
- Responsivness

## Install

- $ git clone project
- $ cd project
- $ npm install
